## Github Repo Clone created using:

-TypeScript
-NextJs
-TailwindCss

## Live Link:

- [Website](https://github-repository-clone-95.netlify.app/)

## To Run Locally:

Go to your terminal, run :

- `git clone https://github.com/Darksied95/Github-repo.git`
- `npm install`
- `npm run dev`
